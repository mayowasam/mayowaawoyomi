A responsive, modern personal portfolio website built with **React**, **Next.js**, and **Tailwind CSS**, featuring:  

- Fully transparent navbar  
- Smooth homepage transitions  
- “Download Resume” button with Google Drive direct download support  
- Mobile-friendly design and animations  

---

## Features

- **Transparent Navbar**: Fully visible over the homepage hero section in both light and dark modes  
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices  
- **Mobile Menu**: Hamburger menu with smooth open/close animations  
- **Clock Component**: Displays current time in the navbar  
- **Download Resume Button**: Downloads PDF directly from Google Drive  
- **Light/Dark Mode Support**: Adapts navbar and text colors for both modes  
- **Smooth Animations**: Hover effects, button scaling, and subtle transitions
